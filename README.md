# AI Response Evaluation Project

## Description

A comparative analysis project focused on evaluating responses generated by multiple Large Language Models (LLMs).

The project includes prompt collection, response comparison, result classification, and quality assessment using structured Excel datasets.

## Key Activities

- Prompt testing and categorization.
- Multi-model response comparison.
- Data collection and organization.
- Quality evaluation and scoring.
- Result documentation and reporting.

## Technologies & Tools

- Microsoft Excel
- Data Analysis
- Large Language Models (LLMs)
- Prompt Engineering
